# Contributing Guidelines

Thank you for your interest in contributing!  To get started:

1. Fork the repository and create a new branch for your feature or fix.
2. Ensure you have Node.js 20 and Terraform 1.8 installed locally.
3. Run `npm ci` to install dependencies and `npm run build` to build the site.
4. Run the tests and linters (if present).
5. Commit your changes with clear messages and open a pull request against `main`.
6. Fill out the pull request template and ensure all CI checks pass.

We welcome bug reports, feature requests, documentation improvements and
other enhancements.
